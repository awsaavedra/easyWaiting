Simple waiting list app for DMV, restaurants, etc.

Technology Stack:
	1. Database = Firebase
	2. Twilio = sending texts to numbers entered
	3. Zapier = automating 
	4. Frontend = AngularJS, HTML, CSS 
	5. Addition: jquery, bootstrap, various firebase libraries (Angularfire, simple login)
